# 🐍 Primeiro Projeto - Atividades em Python

Bem-vindo ao meu primeiro repositório de estudos em Python! Este espaço foi criado para armazenar, organizar e documentar a minha evolução na linguagem, reunindo diversos exercícios, desafios e pequenos projetos práticos.

---

## 🚀 Tecnologias Utilizadas

* **Python 3.x**
* **Git & GitHub** (para controle de versão)
* **IDE:** PyCharm / VS Code (ajuste para a sua)

---

## 📂 Estrutura do Repositório

O projeto está dividido em diferentes níveis e categorias de atividades para facilitar a navegação:

* `01_conceitos_basicos/`: Exercícios com variáveis, operadores matemáticos e entrada/saída de dados.
* `02_estruturas_condicionais/`: Desafios utilizando `if`, `elif` e `else`.
* `03_estruturas_de_repeticao/`: Atividades práticas com laços `for` e `while`.
* `04_funcoes_e_modulos/`: Organização de código em funções reutilizáveis.
* `05_projetos_finais/`: Pequenos programas completos (ex: calculadora, jogo da forca, etc.).
